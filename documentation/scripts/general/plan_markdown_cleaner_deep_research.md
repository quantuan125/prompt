# Implementation Plan - Deep Research Markdown Cleaner

Deep Research Markdown Cleaner is a utility to clean up markdown reports generated by conversion tools (like Deep Research's word2md). The goal is to produce a clean, professional markdown document with normalized citations and a consolidated bibliography.

## User Review Required

> [!IMPORTANT]
> **Citation Consolidation Strategy**: The script will re-number citations sequentially (1, 2, 3...) based on unique URLs. Duplicate URLs cited with different numbers (e.g., [1] and [5] both pointing to google.com) will be merged to the first appearance (both become [1]).
> **Bibliography Generation**: The script will append a `## References` section at the end of the file with the consolidated list of sources.
> **File Output**: The script will generate `*_CLEAN.md` files instead of overwriting, to ensure safety.

## Proposed Changes

### `scripts/general/markdown_cleaner_deep_research.py`

#### [MODIFY] Reference Data Structures
- Add equality/hashing method to `Reference` class to facilitate deduplication (based on URL).

#### [NEW] Phase 4: Consolidation Logic
- `consolidate_references(registry: ReferenceRegistry) -> tuple[ReferenceRegistry, dict[int, int]]`
    - Iterates through the raw registry.
    - Identifies unique URLs.
    - Assigns new sequential IDs (1, 2, 3...) to unique URLs.
    - Creates a mapping of `{old_id: new_id}`.
    - Returns a new clean `ReferenceRegistry` and the renumbering map.

#### [NEW] Phase 5: Text Replacement & Generation
- `apply_renumbering(text: str, mapping: dict[int, int]) -> str`
    - Uses Regex to find `[n]` patterns in the text.
    - Replaces them with `[new_id]` using the mapping.
- `generate_bibliography(registry: ReferenceRegistry) -> str`
    - formats the `ReferenceRegistry` into a markdown list:
      ```markdown
      ## References

      1. [Title Hint](url)
      2. <url>
      ```

#### [NEW] Main Execution Block
- Implement `main()` function:
    - Parse arguments (input files).
    - For each file:
        - Read content.
        - `split_frontmatter`
        - `extract_references` (get body with `[n]` and raw registry)
        - `consolidate_references` (get clean registry and map)
        - `apply_renumbering` (update body to clean numbers)
        - `decode_entities_and_html` (clean HTML artifacts)
        - `generate_bibliography`
        - Combine: Frontmatter + Clean Body + Bibliography.
        - Write to `*_CLEAN.md`.
