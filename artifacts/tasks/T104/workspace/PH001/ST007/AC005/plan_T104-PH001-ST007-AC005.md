# PLAN: T104 (CWS) — Phase 1 / Stream `T104-PH001-ST007` / Activity `AC005`: T102 Directory Migration

## I. EXECUTIVE SUMMARY

**Purpose**: Execute a deterministic, gate-controlled migration of `prompt/artifacts/tasks/T102/**` from mixed legacy roots and mixed epic structures into a `P-STD-004`-conformant layout. AC005 absorbs the live `consultant/` root into canonical T102/T102A/T102B/T102C surfaces, relocates the historical `planner/` root into canonical archive placement, and preserves rollback/evidence discipline through dry-run and post-apply gates.

**Non-goal**: This activity does not amend `P-STD-004` or `P-STD-005`, reopen AC004 scope, or introduce new migration-tool features unless a locked AC005 task explicitly identifies a tool incompatibility that blocks execution.
