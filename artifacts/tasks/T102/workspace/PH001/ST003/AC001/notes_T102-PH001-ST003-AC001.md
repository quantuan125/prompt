Key outcomes:
- **OQ001 resolved**: Epic-level STDs use the same `nnn` number as the source ADR. `T102B-STD-001` → `T102B-STD-001`, consistent with the initiative-level convention.

- **AC001 task structure approved**: 4-batch migration (initiative rename, initiative extract, T102B epic, T102A/C/feature extract) with GATE-001 checkpoint after global reference propagation and before Concept modification.
